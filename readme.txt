# 루틴플로우 (RoutineFlow) 개발 가이드

## 1. 프로젝트 개요

루틴플로우는 사용자의 일상 루틴과 할 일을 효과적으로 관리하는 웹 애플리케이션입니다.

### 핵심 기능
- 루틴 관리 (생성, 수정, 삭제, 완료 체크)
- 할 일 관리 (우선순위, 태그, 마감일)
- 일기 작성
- 통계 및 분석
- 캘린더 뷰

### 기술 스택
- React + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Zustand (상태 관리)
- Lucide React (아이콘)
- Chart.js (통계 차트)
- date-fns (날짜 처리)

## 2. 디자인 시스템

### 색상 팔레트
- Primary: #6366F1 (인디고)
- Secondary: #A78BFA (퍼플)
- Accent: #EC4899 (핑크)
- Success: #10B981 (그린)
- Warning: #F59E0B (옐로우)
- Error: #EF4444 (레드)

### 타이포그래피
- 기본 폰트: Pretendard, Noto Sans KR
- 크기 체계: text-xs부터 text-2xl까지 Tailwind 기본 스케일 사용

### 컴포넌트 스타일링
- 카드: 흰색 배경, 그림자, 둥근 모서리
- 버튼: Primary, Secondary, Accent, Outline 변형
- 입력 필드: 라운드 스타일, 포커스 링 효과
- 뱃지: 상태 표시용 라운드 스타일

## 3. 주요 기능 설명

### 루틴 관리
- 루틴 생성 시 이름, 설명, 시간, 반복 요일, 색상, 아이콘 설정
- 시간이 지난 미완료 루틴에 대한 확인 요청
- 루틴 완료 여부 체크 및 통계 반영

### 할 일 관리
- 우선순위 (높음, 중간, 낮음)
- 태그 시스템
- 마감일 설정
- 필터링 (미완료, 완료, 전체)

### 캘린더
- 월간 보기
- 루틴/할 일 일정 표시
- 날짜별 상세 보기

### 통계
- 일간/주간/월간 완료율
- 차트 시각화
- 요약 정보 대시보드
